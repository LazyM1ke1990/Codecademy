HTML Anatomy


A web page at its basic form is composed and structured by using 'HTML Elements'.
These elements are made up from an 'opening tag' which is the element name enclosed in angled brackets,
'The content' and the 'closing tag' which closes the element. This too is in angled brackets but with a forward slash.
after the opening bracket to distinguish that it's the closing tag and not another opening tag.
